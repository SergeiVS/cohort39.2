Задача: Управление Библиотекой
Описание:

Вы разрабатываете систему для управления библиотекой.
В библиотеке хранятся книги различных жанров, и у каждой книги есть свои экземпляры.
Необходимо создать структуру данных, которая позволяет управлять книгами, авторами и жанрами.

Требования:

    Книги:
        Каждая книга имеет название, автора и жанр.
        Книги хранятся в списке .
        Реализуйте функционал для добавления новой книги в список.

    Авторы:
        Авторы уникальны и хранятся в множестве
        Каждый автор имеет имя и список написанных им книг.
        При добавлении книги, автор добавляется в множество, если его там еще нет.

    Жанры :
        Жанры книг организованы в виде словаря, где ключом является название жанра,
        а значением - список книг этого жанра.
        При добавлении книги, она также добавляется в соответствующий список жанра.

Задачи для реализации:

    Добавление книги:
        Реализуйте метод для добавления новой книги.
        При добавлении книги, автоматически обновляются множество авторов и словарь жанров.

    Поиск книг по автору:
        Реализуйте метод для поиска всех книг заданного автора.

    Поиск книг по жанру:
        Реализуйте метод для получения списка всех книг определенного жанра.

    Удаление книги:
        Реализуйте метод для удаления книги из всех коллекций.

Дополнительные задачи (по желанию):

    Создайте функционал для редактирования информации о книге.
    Реализуйте метод для вывода статистики по библиотеке (количество книг, авторов, жанров).
