Написать приложение, которое описывает книжный шкаф с некоторым количеством полок (больше 2).
На каждой полке может находиться несколько книг.
У книг есть параметры (автор, название)

=======================================

Для создания приложения нам понадобятся:
1. Класс Book - в котором будет храниться информации о каждой книге
2. Класс "книжная полка" - в котором будет храниться данные о наборе книг на одной полке
3. Класс "книжный шкаф" - в котором будет храниться информация о наборе полок

Методы, которые нам понадобятся:

1) создать книгу (заполнить информацией кто автор и т.д.)
2) создать полки, в зависимости от того сколько их будет в нашем шкафу
3) создать шкаф
4) поставить полки в шкаф (добавить полку в шкаф)
5) расставить книги по полкам (добавить книгу на полку)
6) убрать книгу с полки
7) переставить книгу с одной полки на другую (на свободное место)
8) поменять две книги местами
9) отсортировать книги по ... (по автору, по названию)
