Задача 1: Фильтрация и сортировка коллекции

Задание: Дан список строк. Необходимо отфильтровать из этого списка все строки, которые начинаются на букву "A" (без учёта регистра), и вернуть список этих строк, отсортированный в алфавитном порядке.

Цели:

    Применить метод filter для фильтрации строк.
    Использовать sorted для сортировки.
    Конвертировать результат обратно в список с помощью collect(Collectors.toList()).

Задача 2: Группировка элементов и подсчёт количества

Задание: Дан список объектов сотрудников (Employee), где каждый объект содержит имя и отдел, в котором работает сотрудник. Необходимо сгруппировать сотрудников по отделам и подсчитать количество сотрудников в каждом отделе.

Цели:

    Использовать collect(Collectors.groupingBy()) для группировки сотрудников по отделам.
    Применить Collectors.counting() для подсчёта количества сотрудников в каждой группе.

Задача 3: Преобразование списка объектов и агрегация

Задание: Дан список объектов заказов (Order), где каждый заказ содержит список покупок (List<Purchase>) и дату заказа. Каждая покупка содержит название товара и его стоимость. Необходимо рассчитать общую сумму стоимости всех покупок, сделанных в текущем месяце.

Цели:

    Фильтровать заказы по дате, чтобы оставить только те, что сделаны в текущем месяце.
    Использовать flatMap для преобразования Stream<Order> в Stream<Purchase>.
    Применить map для извлечения стоимости покупок.
    Использовать reduce для подсчёта общей суммы стоимости покупок.
