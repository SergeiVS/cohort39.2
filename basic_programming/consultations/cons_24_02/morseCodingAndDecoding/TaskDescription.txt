Написать приложение, которое производит кодировку текстового сообщения в символьную последовательность

Описание решения:
1) Создать структуру данных (коллекцию) в которой будет хранится кодировка (таблица кодировки)
В качестве варианта решения будем использовать HashMap("буква", "последовательность символов") (String, String)

2) Алгоритм работы приложения:
- получить текстовое сообщения для кодировки
- разделить строку на отдельные символы
- закодировать каждый из символов
- собрать закодированные символы в одну "строку"
- вывести эту строку для пользователя

3) Создать сервисы:
- inputMessage();
- splitMessageForSymbols();
- codingSymbols();
- createCodedMessage();
- outputCodedMessage();
