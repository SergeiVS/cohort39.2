1. Создайте робота (с использованием соответствующего класса)
2. у Робота должна быть марка
3. Научите вашего робота здороваться и представляться
4. Научим робота конвертировать валюту.
Для этого робот должен получить информацию сколько у нас долларов
 и ответить сколько это будет евро.
 Курс конвертации робот знает сам.
