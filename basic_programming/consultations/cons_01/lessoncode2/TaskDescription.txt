Написать приложение, которое будет учитывать наличие товаров на складе:
1) приложение должно работать с тремя категориями товара:
- телефон
- компьютер
- принтер
2) приложение должно запрашивать у пользователя количество каждого типа товара
3) запрашивать характеристики каждого товара по типам
4) вывести всю информацию о складе


------------------------------------------------------
Все ваше приложение будет состоять из трех частей:
1) общение с пользователем и запрос данных
 либо рандомная генерация данных
 -----
 2) непосредственно обработка данных
 -----
 3) вывод итоговой информации на экран
