Написать приложение, которое осуществляет подсчет количество вхождений слов в текст, т.е.
дан текст и необходимо:
1) составить список слов, которые в этот текст входит
2) подсчитать количество вхождений каждого слова
3) найти самое редкое слово и самое часто встречающееся слово

Для хранения данных нам понадобятся:
1) переменная, где будет храниться сам текст
2) все слова этого текста -> коллекция для хранения: ArrayList<String>
3) для списка слов (без повторений) -> коллекция Set<String>
4) для подсчета количества вхождений -> коллекция Map<String, Integer>
5) для хранения самых редких и самых часто встречающихся слов -> ArrayList<String> и переменная Integer

Сервисы:
1) получение текста
2) разделение текста на отдельные слова - как результат данные из п.2
3) заполнение данных п.3
4) подсчет количества повторений -> данные п.4
5) поиск самых редких и самых распространенных слов в тексте -> данные п.5
