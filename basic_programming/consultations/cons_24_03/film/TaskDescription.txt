У нас есть класс "Film" со следующими полями
- String title ( название фильма)
- String genre (жанр)
- String director (режиссер)

Ваша задача написать приложение, которая выполняет следующие действия
1) создает список
2) добавляет в список новый фильм
3) находит и выводит все фильмы по жанру
4) находит и выводит все фильмы по режиссеру
